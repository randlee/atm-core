# `atm clear`

CLI ownership for `atm clear`:

- clear-mode flag parsing
- actor override parsing
- conversion into `atm-core` clear requests
- dry-run rendering
- human-readable output
- JSON output

Clear eligibility remains owned by `atm-core`.
